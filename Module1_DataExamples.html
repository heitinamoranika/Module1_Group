Overview

This document provides additional data that you can use to test our your code. These data looks at outbound network traffic for specific ip addresses from my local router on September 7, 2020. While it is not server data, the data below closely mimics the type of server analytic data for module 1.

Example 1: Youtube Internet Traffic From My Router

I’ve collected all the outbound traffic data on my local router for a span of 30 minutes. Specifically, I was tracking what I think is one of Youtube’s ip address, but it might be something completely different; most likely, the ip address(es) that I was tracking is the content delivery network that Youtube uses to deliver content to my local router.

\(y_t\) is measured in kilobytes and represents the cumulative outbound traffic. \(t\) is measured in seconds and spans from 1 to 1000. In the plot below, I’ve set the maxcap to \(100 \cdot 10^3\) and the router hit that limit at \(t = 1501\). Obviously, you want to aim below that (but not by too much) as overshooting the time would be much worse than undershooting it.

maxcap = 100*10^3;
out_data = read.csv("out_youtube.csv",header=TRUE)
plot(out_data$t,out_data$y_t,ylim=c(0,maxcap),xlim=c(0,1600),
     ylab="Youtube Traffic (Kb)",xlab="Time (sec)",main="Youtube Traffic Data")
abline(h = maxcap,col="red",lwd=2)

Example 2: Box Sync Data

Also in the span of 30 minutes, I’ve also examined the traffic going in and out of the port that Box uses to sync my local folders. To simulate a server outage, for a brief period lasting about 15 seconds-ish, I blocked the port that Box uses and killed the Box process on my computer; that’s why, you’ll see a missing set of values.

Again, \(y_t\) is measured in kilobytes and represents the cumulative traffic. \(t\) is measured in seconds and spans from 1 to 306. In the plot below, I’ve set the maxcap to \(100\) and I would say that good timepoint to alert the engineer of hard drive capacity is slightly below 300-ish.

maxcap=100
box_data = read.csv("out_box.csv",header=TRUE)
plot(box_data$t,box_data$y_t,ylim=c(-10,maxcap),xlim=c(0,400),
     ylab="Box Traffic (kb) ",xlab="Time (sec)",main="Box Traffic Data")
abline(h = maxcap,col="red",lwd=2)
